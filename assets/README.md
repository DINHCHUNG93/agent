# Brand assets

The README references `assets/logo.png`.

Save the PentesterFlow logo here as **`logo.png`** (transparent background, the
wide hooded-shield + wordmark lockup). The README embeds it at 520px wide.

```sh
# from the file you have locally, e.g.:
cp /path/to/pentesterflow-logo.png assets/logo.png
git add assets/logo.png && git commit -m "docs: add project logo" && git push
```
